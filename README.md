# 🏛️ Casa de Aurayen

> “Todo viajante que cruza estas páginas deixa de ser apenas um leitor. Torna-se um peregrino em busca do próprio aperfeiçoamento.”

## Sobre

A Casa de Aurayen é o portal oficial do universo de Aurayen.

Este projeto não é apenas um site de leitura, mas uma experiência imersiva de exploração de um mundo fictício.

---

## Objetivo

- Centralizar o universo de Aurayen
- Publicar a jornada de Shinsei Hoshino
- Apresentar crônicas, mapas, raças e facções
- Criar uma experiência de leitura imersiva
- Servir como base técnica e criativa do universo

---

## Tecnologias
