# 🧭 Filosofia

Aurayen é um universo narrativo.

Este projeto segue três pilares:

## 1. Conteúdo primeiro

A história sempre vem antes da interface.

## 2. Simplicidade

Soluções simples são preferidas.

## 3. Imersão

Cada página deve parecer parte de um mundo vivo.

---

## Regra principal

Se uma decisão técnica conflita com a experiência do visitante,
a experiência sempre vence.
