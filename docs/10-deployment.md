# 🚀 Deploy

## Objetivo

Garantir que a Casa de Aurayen seja publicada de forma simples, estável e reproduzível.

---

## Plataforma

O projeto será hospedado no GitHub Pages.

---

## Estratégia de build

O site é totalmente estático.

O processo de deploy consiste em:

1. build do projeto
2. geração de arquivos estáticos
3. publicação da pasta de saída

---

## Comandos padrão

```bash
npm install
npm run build
npm run preview
```
