# Direção de Arte — A Casa de Aurayen

> "A Casa de Aurayen não é um website. Ela é um mundo."

---

# Objetivo

A Casa de Aurayen deve proporcionar a sensação de atravessar um limite entre o mundo real e um universo antigo, misterioso e vivo.

O visitante não deve sentir que abriu uma página da internet.

Ele deve sentir que entrou em um lugar que existe independentemente dele.

Toda decisão de design, interface e interação deve reforçar essa sensação.

---

# Filosofia

## O mundo vem antes da interface.

A interface nunca deve parecer uma camada colocada sobre o cenário.

Ela faz parte do mundo.

O usuário não utiliza menus.

Ele explora ambientes.

O usuário não abre páginas.

Ele atravessa portas.

O usuário não clica em botões.

Ele interage com objetos.

---

# Experiência desejada

Ao entrar em Aurayen, o visitante deve experimentar as seguintes emoções:

1. Curiosidade
2. Descoberta
3. Fascínio
4. Pertencimento

O objetivo não é impressionar pela tecnologia.

O objetivo é despertar a sensação de estar em um lugar real.

---

# Princípios de Design

## Revelar em vez de mostrar

Aurayen nunca apresenta todas as informações imediatamente.

O visitante deve descobrir o mundo aos poucos.

---

## Explorar em vez de navegar

A estrutura do projeto deve lembrar um mundo explorável.

Evitar layouts tradicionais com grandes páginas e rolagem infinita.

Sempre que possível, a navegação deve acontecer através de ambientes.

---

## Interagir em vez de clicar

Toda interação deve possuir significado dentro do universo.

Exemplos:

- abrir uma porta
- acender uma tocha
- desenrolar um pergaminho
- consultar um mapa
- abrir um livro
- ativar um selo

Nunca utilizar elementos genéricos quando houver uma alternativa narrativa.

---

# Identidade Visual

Aurayen deve transmitir a sensação de uma construção antiga.

Referências:

- bibliotecas medievais
- mosteiros
- castelos
- templos antigos
- academias de conhecimento

Evitar aparência futurista.

Evitar aparência corporativa.

Evitar aparência minimalista moderna.

---

# Materiais

Priorizar materiais naturais.

- pedra
- madeira
- ferro
- bronze
- couro
- pergaminho
- vidro antigo

Nada deve parecer plástico.

Nada deve parecer excessivamente digital.

---

# Paleta de Cores

Predominância de:

- madeira escura
- pedra
- dourado envelhecido
- bronze
- pergaminho
- azul profundo
- verde musgo

Evitar:

- branco puro
- preto absoluto
- neon
- gradientes artificiais

---

# Iluminação

A iluminação deve parecer natural.

Fontes de luz:

- tochas
- velas
- lareiras
- janelas
- luar

Sempre utilizar contraste.

Nunca iluminação totalmente uniforme.

---

# Tipografia

Os títulos podem possuir aparência clássica.

O corpo do texto deve priorizar leitura confortável.

Evitar fontes extremamente ornamentadas em textos longos.

---

# Interface

A interface deve desaparecer.

Sempre que possível substituir componentes tradicionais.

| Interface comum | Aurayen |
|----------------|----------|
| Botão | Brasão |
| Menu | Mapa da Casa |
| Lista | Estante |
| Card | Livro |
| Modal | Pergaminho |
| Link | Porta |
| Página | Ambiente |

---

# Ambientes

A Casa é composta por ambientes.

Cada ambiente possui personalidade própria.

## Salão Principal

Centro da Casa.

Hub de navegação.

O visitante retorna para ele durante sua jornada.

---

## Biblioteca

Local dos contos.

O visitante consulta livros.

Não listas.

---

## Arquivos

Cronologias.

Mapas.

Relíquias.

Bestiário.

Documentos.

---

## Salão dos Guardiões

Personagens.

Histórias.

Ilustrações.

Evolução.

---

## Observatório

Cosmologia.

Constelações.

Origem da magia.

---

## Pátio

Ambiente contemplativo.

Pouca interação.

Muito silêncio.

---

# Movimento

Toda animação deve possuir propósito.

Evitar movimentos exagerados.

Exemplos:

✔ Porta pesada abrindo lentamente.

✔ Poeira atravessando um feixe de luz.

✔ Chama oscilando.

✔ Pergaminho desenrolando.

✔ Livro abrindo.

Evitar:

❌ animações chamativas

❌ elementos pulando

❌ efeitos excessivos

---

# Som

Aurayen utiliza sons ambientes.

Não música constante.

Exemplos:

- vento
- fogo
- páginas
- madeira
- passos
- sinos
- chuva

O silêncio faz parte da experiência.

---

# Contos

Os contos não devem parecer artigos.

Cada conto deve ser apresentado como um documento preservado.

Características:

- textura de pergaminho
- marcas do tempo
- bordas desgastadas
- selo de Aurayen
- tipografia elegante

---

# Arquitetura Técnica

A implementação deve priorizar tecnologias nativas.

Fase inicial:

- Astro
- React
- Tailwind
- CSS Animations
- JavaScript

Adicionar bibliotecas externas apenas quando necessário.

---

# O que evitar

Não transformar Aurayen em:

- landing page
- dashboard
- portfólio tradicional
- site institucional
- blog comum

Sempre que uma decisão gerar dúvida, perguntar:

> "Isso parece pertencer ao mundo de Aurayen?"

Se a resposta for "não", a solução deve ser repensada.

---

# Objetivo Final

O visitante deve esquecer que está utilizando um navegador.

Ao sair da Casa de Aurayen, a sensação desejada é:

> "Eu não visitei um site.
>
> Eu explorei um lugar."
