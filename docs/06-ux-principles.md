# 🧠 Princípios de UX

## Objetivo

Criar sensação de imersão e descoberta.

---

## Regras

- Interface nunca deve competir com o conteúdo
- Navegação deve ser intuitiva e silenciosa
- Espaço em branco é parte do design
- Animações devem ser sutis
- O usuário nunca deve se sentir pressionado

---

## Experiência desejada

- leitura contemplativa
- exploração natural
- curiosidade crescente
