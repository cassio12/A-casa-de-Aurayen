# 🏗️ Arquitetura

O projeto é um site estático gerado com Astro.

---

## Estrutura

- `/pages` → rotas
- `/components` → blocos reutilizáveis
- `/layouts` → estrutura visual
- `/content` → Markdown do universo
- `/docs` → regras do sistema

---

## Regra

Conteúdo nunca deve depender de lógica complexa.
